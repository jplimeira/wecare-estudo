## 📌 Sobre o projeto

O **We Care** é uma página web desenvolvida com foco em **interface moderna**, **organização de código** e **responsividade**.

O projeto simula uma landing page voltada para cuidados com pets, apresentando uma estrutura limpa, visual agradável e boa experiência para o usuário.

## 💻 HTML - Estrutura

O projeto utiliza uma estrutura semântica e organizada:

- `<header>` → Menu de navegação
- `<main>` → Conteúdo principal
- `<section>` → Área de apresentação
- `<img>` → Ilustração principal
- `<h1>` e `<h2>` → Títulos
- `<p>` → Descrição
- `<button>` → Call to Action

### ✔ Boas práticas aplicadas:

- Uso de `<!DOCTYPE html>`
- `lang="pt-br"`
- `meta viewport` para responsividade 📱
- Separação de CSS externo
- Código limpo e legível

---

## 🎨 CSS - Estilização

O CSS foi utilizado para criar uma interface moderna e agradável:

- Reset básico com `*`
- Tipografia com **Montserrat**
- Uso de cores suaves e contrastes
- Botões com **hover e active**
- Alinhamento com `inline-block`
- Uso de `margin` e `text-align`

### ✨ Destaques:

- Efeitos de interação (`hover` e `active`)
- Botões com design moderno
- Layout visual equilibrado

---

## 📱 Responsividade

O projeto foi desenvolvido com foco em diferentes tamanhos de tela:

### 🔹 Mobile (até 900px)
- Layout adaptado para coluna
- Imagem centralizada
- Redução de fontes
- Melhor leitura em telas menores

### 🔹 Tablet (901px até 1100px)
- Ajustes de espaçamento
- Redimensionamento de textos
- Melhor aproveitamento de tela

### 🔹 Desktop
- Layout horizontal
- Separação entre imagem e conteúdo
- Maior destaque visual
